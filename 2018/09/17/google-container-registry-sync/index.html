дёҖгҖҒиө·еӣ 

зҺ© Kubenretes зҡ„еҹәжң¬йғҪеҫҲжё…жҘҡпјҢKubernetes еҫҲеӨҡз»„д»¶зҡ„й•ңеғҸе…ЁйғЁжүҳз®ЎеңЁ gcr.io иҝҷдёӘеҹҹеҗҚдёӢ(зҺ°еңЁжҚўжҲҗдәҶ k8s.gcr.io)пјӣз”ұдәҺдј—жүҖе‘ЁзҹҘзҡ„еҺҹеӣ пјҢиҝҷдёӘзҪ‘з«ҷеңЁеӣҪеҶ…жҳҜдёҚеҸҜиҫҫзҡ„пјӣеҪ“ж—¶з”ұдәҺ Docker Hub жҸҗдҫӣдәҶ Auto Build еҠҹиғҪпјҢжңәжҷәзҡ„жғіеҲ°дёҖдёӘи§ЈеҶіеҠһжі•пјӣе°ұжҳҜеҲ©з”Ё Docker Hub зҡ„ Auto BuildпјҢеҲӣе»әеҸӘжңүдёҖиЎҢзҡ„ DockerfileпјҢйҮҢйқўе°ұдёҖеҸҘ FROM gcr.io/xxxxпјҢз„¶еҗҺи®© Docker Hub её®дҪ жһ„е»әе®ҢжҲҗеҗҺжӢүеҸ–еҚіеҸҜ

иҝҷз§ҚеҘ—и·Ҝзҡ„еҹәжң¬ж–№жЎҲе°ұжҳҜеҲ©з”ЁдёҖдёӘз¬¬дёүж–№е…¬е…ұд»“еә“пјҢиҝҷдёӘд»“еә“еҸҜд»Ҙи®ҝй—®дёҚеҸҜиҫҫзҡ„ gcr.ioпјҢз„¶еҗҺз”ҹжҲҗй•ңеғҸпјҢжҲ‘д»¬еҶҚд»ҺиҝҷдёӘд»“еә“ pull еҚіеҸҜпјӣдёәжӯӨжҲ‘еҲӣе»әдәҶдёҖдёӘ Github д»“еә“(docker-library)пјӣж—¶йҡ”иҝҷд№Ҳд№…д»ҘеҗҺпјҢжҲ‘зҢңжғіеӨ§е®¶йғҪе·Із»ҸжңүдәҶиҝҷз§ҚиҮӘе·ұзҡ„д»“еә“вҖҰдёҚиҝҮжңҖиҝ‘еҸ‘зҺ°иҝҷдёӘд»“еә“д»Қз„¶еңЁжңүдәә forkвҖҰ

дёәдәҶдёҖеҠіж°ёйҖёзҡ„и§ЈеҶіиҝҷдёӘй—®йўҳпјҢеҸӘиғҪж’ёзӮ№д»Јз Ғи§ЈеҶіиҝҷдёӘй—®йўҳдәҶ

дәҢгҖҒд»“еә“дҪҝз”Ё

дёәдәҶи§ЈеҶідёҠиҝ°й—®йўҳпјҢжҲ‘еҶҷдәҶдёҖдёӘ gcrsync е·Ҙе…·пјҢе№¶дё”еҖҹеҠ© Travis CI и®©е…¶жҜҸеӨ©иҮӘеҠЁиҝҗиЎҢпјҢе°ҶжүҖжңүз”Ёеҫ—еҲ°зҡ„ gcr.io дёӢзҡ„й•ңеғҸеҗҢжӯҘеҲ°дәҶ Docker Hub

зӣ®еүҚеҜ№дәҺдёҖдёӘ gcr.io дёӢзҡ„й•ңеғҸпјҢеҸҜд»ҘзӣҙжҺҘжӣҝжҚўдёә gcrxio з”ЁжҲ·еҗҚпјҢз„¶еҗҺд»Һ Docker Hub зӣҙжҺҘжӢүеҸ–пјҢд»ҘдёӢдёәдёҖдёӘзӨәдҫӢ:

# еҺҹе§Ӣе‘Ҫд»Ө
docker pull k8s.gcr.io/kubernetes-dashboard-amd64:v1.10.0

# дҪҝз”ЁеҗҢжӯҘд»“еә“
docker pull gcrxio/kubernetes-dashboard-amd64:v1.10.0

дёүгҖҒеҗҢжӯҘз»ҶиҠӮиҜҙжҳҺ

дёәдәҶдҝқиҜҒеҗҢжӯҘй•ңеғҸзҡ„е®үе…ЁжҖ§пјҢеҗҢжӯҘе·Ҙе…·е·Із»ҸејҖжәҗеңЁ gcrsync д»“еә“пјҢеҗҢжӯҘз»ҶиҠӮеҰӮдёӢ:

  • е·Ҙе…·жҜҸеӨ©з”ұ Travis CI иҮӘеҠЁиҝӣиЎҢдёҖж¬Ў buildпјҢз„¶еҗҺиҝӣиЎҢжҺЁйҖҒ
  • е·Ҙе…·жҜҸж¬ЎжҺЁйҖҒеүҚйҰ–е…Ҳ clone е…ғж•°жҚ®д»“еә“ gcr
  • е·Ҙе…·жҜҸж¬ЎжҺЁйҖҒйҰ–е…ҲиҺ·еҸ– gcr.io жҢҮе®ҡ namespace дёӢзҡ„жүҖжңүй•ңеғҸ(namesapce з”ұ .travis.yml script ж®өе®ҡд№ү)
  • иҺ·еҸ– gcr.io й•ңеғҸеҗҺпјҢеҶҚиҜ»еҸ–е…ғж•°жҚ®д»“еә“(gcr) дёӯдёҺ namesapce еҗҢеҗҚж–Үд»¶(е®һйҷ…жҳҜдёӘ json)
  • жҺҘзқҖеҜ№жҜ”еҸҢж–№е·®ејӮпјҢеҫ—еҮәйңҖиҰҒеҗҢжӯҘзҡ„й•ңеғҸ
  • жңҖеҗҺйҖҡиҝҮ API и°ғз”Ёжң¬ең°зҡ„ docker иҝӣиЎҢ pullгҖҒtagгҖҒpush ж“ҚдҪңпјҢе®ҢжҲҗй•ңеғҸжҺЁйҖҒ
  • жүҖжңүй•ңеғҸжҺЁйҖҒжҲҗеҠҹеҗҺпјҢжӣҙж–°е…ғж•°жҚ®д»“еә“еҶ… namespace еҜ№еә”зҡ„ json ж–Үд»¶пјҢжңҖеҗҺеңЁз”ҹжҲҗ CHANGELOGпјҢжү§иЎҢ git push еҲ°иҝңзЁӢе…ғж•°жҚ®д»“еә“

з»јдёҠжүҖиҝ°пјҢеҰӮжһңжғіеҫ—зҹҘе…·дҪ“ gcrxio з”ЁжҲ·дёӢйғҪжңүйӮЈдәӣй•ңеғҸпјҢеҸҜзӣҙжҺҘи®ҝй—® gcr е…ғж•°жҚ®д»“еә“пјҢжҹҘзңӢеҜ№еә” namesapce еҗҢеҗҚзҡ„ json ж–Үд»¶еҚіеҸҜпјӣжҜҸеӨ©еўһйҮҸеҗҢжӯҘзҡ„дҝЎжҒҜдјҡиҝҪеҠ еҲ° gcr д»“еә“зҡ„ CHANGELOG.md ж–Үд»¶дёӯ
